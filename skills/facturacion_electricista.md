# Skill: Facturación para Electricista de Hogar (Baja Tensión)

## Descripción

Esta habilidad define cómo Apolonio Zánjales ayuda a generar y revisar facturas para clientes de hogar.

## Funciones

- Calcular importe total de la factura.
- Aplicar IVA correspondiente.
- Incluir datos del autónomo (NIF, nombre, dirección).
- Incluir datos del cliente.
- Describir trabajos realizados (mano de obra, materiales).
- Generar texto claro y profesional.

## Ejemplo de uso

Entrada:
- Cliente: Juan Pérez
- Trabajo: Instalación de puntos de luz en salón
- Mano de obra: 3 horas
- Materiales: 4 focos LED

Salida:
- Factura detallada lista para enviar al cliente.
